# slmaker - Small Language Model Maker

밑바닥부터 만드는 SLM(Small Language Model) 제작기입니다. / This is a journey to create a Small Language Model (SLM) from scratch.

## 개요 / Overview
이 프로젝트는 언어 모델의 핵심 원리를 이해하고 최적화된 SLM을 구축하는 것을 목표로 합니다. / This project aims to understand the core principles of language models and build an optimized SLM.
